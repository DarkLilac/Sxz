Text Flow Around Sxz Image

Since a Sxz image need not be rectangular, text can be drawn around an image in a non-rectangular fashion. This demo is inpsired by CSS Shapes editor. The distinction is that paths are not required since Sxz has an irregular border once the white background of the image below was removed when it was converted from PNG to Sxz. This demo is limited owing to a lack of completeness for laying out text and drawing on the Canvas element. It's assumed that a web browser's first class support for Sxz would render better.
